# 🎨 智能照片自动布局指南

您的网站现在支持**智能照片检测和自动布局**！系统会自动检测文件夹中的照片数量，并选择最合适的显示方式。

---

## ✨ 功能特点

### 自动检测
- ✅ 自动扫描文件夹中的照片
- ✅ 支持多种命名规则（award1, photo1, tdk1, medal1 等）
- ✅ 支持多种图片格式（.jpg, .jpeg, .png, .webp）
- ✅ 智能跳过缺失的照片

### 自动布局
系统根据检测到的照片数量自动选择最佳布局：

| 照片数量 | 布局样式 | 视觉效果 |
|---------|---------|---------|
| **0张** | 占位符 | 显示"No photos available" |
| **1张** | 单图展示 | 大尺寸单张照片 |
| **2张** | 左右分割 | 两张照片并排显示 |
| **3张** | 拼贴布局 | 1张大图 + 2张小图 |
| **4张** | 2×2网格 | 四宫格布局 |
| **5张+** | 轮播图 | 自动切换 + 手动导航 |

## 📸 详细布局说明

### 1️⃣ 单张照片布局

**适用情况：** 只有 1 张照片

**效果：**
```
┌─────────────────────┐
│                     │
│     单张大图        │
│     (4:3比例)       │
│                     │
└─────────────────────┘
```

**文件示例：**
```
awards/2025/
└── award1.jpg
```

---

### 2️⃣ 两张照片布局（左右分割）

**适用情况：** 有 2 张照片

**效果：**
```
┌──────────┬──────────┐
│          │          │
│  照片1   │  照片2   │
│          │          │
└──────────┴──────────┘

**文件示例：**

```

└── award3.jpg     ← 会被检测到

## 🆘 常见问题

### Q1: 照片没有显示怎么办？

**A:** 检查以下几点：
1. **文件路径是否正确**
   ```
   ✅ 正确：images/awards/2025/award1.jpg
   ❌ 错误：images/2025/award1.jpg
   ❌ 错误：awards/2025/award1.jpg
   ```

2. **文件名是否正确**
   - 区分大小写（award1 ≠ Award1）
   - 不包含空格（`award 1.jpg` 是错误的）
   - 扩展名正确（.jpg 不是 .JPG）

3. **浏览器缓存**
   - 按 `Ctrl + F5` 强制刷新
   - 或按 `F12` → Network → 勾选 "Disable cache"

4. **查看控制台错误**
   - 按 `F12` 打开开发者工具
   - 查看 Console 选项卡是否有错误信息

### Q2: 显示"Loading photos..."一直不消失

**A:** 可能原因：
1. **照片文件不存在** - 检查文件夹中是否有照片
2. **网络问题** - 检查图片路径是否正确
3. **JavaScript错误** - 按 `F12` 查看控制台错误

解决方案：
```bash
# 确保至少有一张照片存在
images/awards/2025/award1.jpg
```

### Q3: 照片显示变形怎么办？

**A:** CSS 已设置 `object-fit: cover` 自动裁剪，如果还是变形：
- 使用 4:3 或 16:9 比例的横向照片
- 使用图片编辑工具裁剪成合适比例
- 推荐尺寸：1200×900 (4:3)

### Q4: 可以只放1张或2张照片吗？

**A:** 可以！系统会自动适应：
- 1张 → 单图展示
- 2张 → 左右分割
- 3张 → 拼贴布局
- 4张 → 2×2网格
- 5+张 → 轮播图

### Q5: 轮播图可以调整速度吗？

**A:** 可以！打开 `script.js` 找到：
```javascript
}, 5000); // 5秒自动切换
```
改成：
```javascript
}, 3000); // 3秒自动切换
```

### Q6: 能否禁用自动轮播？

**A:** 可以！在 `script.js` 中注释掉轮播启动代码：
```javascript
// 注释这一段，禁用自动轮播
/*
if (photos.length >= 5) {
    const carousel = gallery.querySelector('.photo-carousel');
    if (carousel) {
        setInterval(() => {
            const nextBtn = carousel.querySelector('.carousel-btn.next');
            if (nextBtn) {
                changeSlide(nextBtn, 1);
            }
        }, 5000);
    }
}
*/
```

### Q7: 照片太多，加载很慢怎么办？

**A:** 优化建议：
1. **压缩照片** - 使用 [TinyPNG.com](https://tinypng.com/)
2. **限制数量** - 每个年份最多 5-6 张
3. **降低分辨率** - 1200×900 已足够

### Q8: 想要不同年份使用不同布局

**A:** 系统会自动根据照片数量选择布局：
- 2025年放3张 → 自动使用拼贴
- 2024年放5张 → 自动使用轮播
- 2023年放1张 → 自动使用单图
