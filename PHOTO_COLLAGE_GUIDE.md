## 🔧 如何创建文件夹并上传

### Windows 用户：

**步骤 1: 创建文件夹**
```bash
1. 打开文件总管，进入项目文件夹
   C:\Users\ASUS\PycharmProjects\TingLun-SU_Allen\

2. 右键 → 新建 → 文件夹 → 命名为 "images"

3. 进入 images 文件夹，创建 "awards" 文件夹

4. 进入 awards 文件夹，创建以下4个文件夹：
   - 2025
   - 2024
   - 2023
   - early
```

**步骤 2: 准备照片**
1. 收集每个年份的最佳照片（每年3张）
2. 重命名照片：
   - 2025 年：`award1.jpg`, `award2.jpg`, `award3.jpg`
   - 2024 年：`award1.jpg`, `award2.jpg`, `award3.jpg`
   - 2023 年：`tdk1.jpg`, `tdk2.jpg`, `tdk3.jpg`
   - 早期：`medal1.jpg`, `medal2.jpg`, `medal3.jpg`
3. 使用 [TinyPNG.com](https://tinypng.com/) 压缩照片（< 500KB）

**步骤 3: 上传照片**
1. 将每个年份的3张照片复制到对应文件夹
2. 确认文件名完全正确（区分大小写）

**步骤 4: 上传个人头像**
1. 准备一张正方形的个人照片
2. 重命名为 `profile.jpg`
3. 放入 `images/` 文件夹（不是 awards 文件夹）

**步骤 5: 刷新查看**
1. 打开 `index.html`
2. 按 `Ctrl + F5` 强制刷新
3. 查看拼贴效果！

## 💡 专业建议

### 大图照片选择技巧：
1. **优先选择"举起奖杯"的照片** - 动作有力量感
2. **确保主体清晰** - 您的脸部和奖杯都要清楚
3. **背景简洁** - 避免杂乱，突出主题
4. **光线充足** - 避免逆光或过暗
5. **表情自然** - 自信的笑容最佳

### 小图照片选择技巧：
- **小图 1**: 选择能"证明"获奖的照片（证书、奖状）
- **小图 2**: 选择能"补充故事"的照片（团队、现场）
- 避免重复内容（不要两张都是证书）
- 确保清晰可见（即使缩小也能看清）

### 照片风格统一：
- 同一年份的3张照片尽量来自同一活动
- 色调和光线相近
- 避免一张室内一张室外的巨大反差

## 🆘 常见问题

### Q1: 只有 1-2 张照片怎么办？
**A:** 可以重复使用或使用相关照片：
- 只有 1 张：用这张做大图，小图可以用证书扫描件或活动海报
- 只有 2 张：其中一张做大图，另一张分割成两个小图或重复使用

### Q2: 照片不显示怎么办？
**A:** 检查：
1. 文件名是否完全正确（包括大小写、扩展名）
2. 照片是否在正确的文件夹中
3. 按 `F12` 打开浏览器控制台查看错误
4. 确认照片文件不是 0KB（损坏的文件）

### Q3: 照片显示变形怎么办？
**A:**
- CSS 已设置 `object-fit: cover`，会自动裁剪
- 如果还是变形，请使用 4:3 比例的横向照片
- 可以用图片编辑工具裁剪成 4:3 比例

### Q4: 可以不放满 3 张照片吗？
**A:** 可以，但建议至少 2 张：
- 只放大图：删除 `<div class="grid-side">` 整个区块
- 放 1 大 1 小：只保留一个小图的 `<img>` 标签

### Q5: 想要 4 张照片怎么办？
**A:** 需要修改 HTML 和 CSS，我可以帮您实现：
```
┌─────────┬────┬────┐
│         │ 1  │ 2  │
│  大图   ├────┼────┤
│         │ 3  │ 4  │
└─────────┴────┴────┘
```
