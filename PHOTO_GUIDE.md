# 📸 照片上传完整指南

这份指南将告诉您如何准备和上传所有需要的照片。

## 📂 第一步：创建文件夹结构

在项目根目录（`TingLun-SU_Allen/` 文件夹）中，创建以下文件夹：

```
TingLun-SU_Allen/
├── index.html
├── style.css
├── script.js
├── images/                          ← 手动创建这个文件夹
│   ├── profile.jpg                 ← 个人头像照片
│   ├── awards/                     ← 手动创建这个子文件夹
│   │   ├── 2025-national-competition.jpg
│   │   ├── 2024-automation-award.jpg
│   │   ├── 2023-tdk-cup.jpg
│   │   └── early-achievements.jpg
│   └── projects/                   ← （可选）项目展示照片
```

### Windows 中创建文件夹的方法：

1. 打开文件总管，进入 `C:\Users\ASUS\PycharmProjects\TingLun-SU_Allen\`
2. 右键 → 新建 → 文件夹 → 命名为 `images`
3. 进入 `images` 文件夹
4. 再创建一个新文件夹 → 命名为 `awards`

## 🖼️ 第二步：准备照片

### 1. **个人头像照片** (`images/profile.jpg`)

**用途：** 显示在 Overview 部分右侧的个人资料卡片中

**要求：**
- 文件名：必须是 `profile.jpg` 或 `profile.png`
- 尺寸：建议 400x400 或更大（会自动缩放到 200x200）
- 比例：**正方形**（1:1）最佳
- 格式：JPG 或 PNG
- 建议：专业证件照风格，浅色或白色背景

**如何拍摄/准备：**
- ✅ 正装或商务休闲装
- ✅ 背景干净（白色、浅灰色或浅蓝色）
- ✅ 光线充足，面部清晰
- ✅ 可以使用 LinkedIn 风格的职业照

**放置位置：**
```
images/profile.jpg
```

---

### 2. **奖项照片** (`images/awards/`)

这些照片会显示在 Awards 部分的右侧，让您的获奖经历更生动。

#### 📁 `2025-national-competition.jpg`
- **内容建议：** 2025年全国职业技术学院竞赛的颁奖现场、奖状、奖杯或团队合照
- **尺寸：** 800x600 或 1200x900（4:3 比例）
- **格式：** JPG

#### 📁 `2024-automation-award.jpg`
- **内容建议：** 2024年自动化设备发明奖的照片（可以是多个奖项的合集）
- **尺寸：** 800x600 或 1200x900（4:3 比例）
- **格式：** JPG

#### 📁 `2023-tdk-cup.jpg`
- **内容建议：** TDK杯竞赛的照片、作品展示或颁奖场景
- **尺寸：** 800x600 或 1200x900（4:3 比例）
- **格式：** JPG

#### 📁 `early-achievements.jpg`
- **内容建议：** 2020-2022年早期竞赛的奖牌展示、奖状拼图或竞赛现场照片
- **尺寸：** 800x600 或 1200x900（4:3 比例）
- **格式：** JPG

**放置位置：**
```
images/awards/2025-national-competition.jpg
images/awards/2024-automation-award.jpg
images/awards/2023-tdk-cup.jpg
images/awards/early-achievements.jpg
```

## 🎨 第三步：照片处理建议

### 如果照片太大，可以使用这些免费工具压缩：

**在线工具（推荐）：**
1. **TinyPNG** - https://tinypng.com/
   - 拖拽上传照片
   - 自动压缩（通常可减少 50-70% 大小）
   - 下载压缩后的文件

2. **Squoosh** - https://squoosh.app/
   - Google 开发的在线图片压缩工具
   - 可以调整质量和尺寸

**Windows 内置工具：**
- 右键照片 → 打开方式 → 相片 → 右上角"..."→ 调整大小

**目标文件大小：**
- 个人头像：< 200KB
- 奖项照片：< 500KB 每张

## 📤 第四步：上传照片

### 方法 1：直接复制粘贴（最简单）

1. 准备好您的照片
2. 重命名照片为对应的文件名（例如：`profile.jpg`）
3. 复制照片
4. 粘贴到对应的文件夹中

### 方法 2：拖拽上传

1. 在 VS Code 中打开项目
2. 左侧文件列表中找到 `images` 文件夹
3. 直接将照片拖拽到文件夹中
4. 右键重命名为正确的文件名

## ✅ 第五步：检查照片是否正确显示

### 1. 检查文件路径
确保文件结构如下：
```
✓ images/profile.jpg
✓ images/awards/2025-national-competition.jpg
✓ images/awards/2024-automation-award.jpg
✓ images/awards/2023-tdk-cup.jpg
✓ images/awards/early-achievements.jpg
```

### 2. 检查文件名
- ⚠️ 文件名必须**完全一致**（包括大小写、破折号、扩展名）
- ⚠️ 不能有空格（用 `-` 代替空格）
- ⚠️ 扩展名必须是 `.jpg` 或 `.png`（小写）

### 3. 刷新浏览器查看
1. 保存所有文件
2. 打开 `index.html`
3. 按 `Ctrl + F5` 强制刷新
4. 检查照片是否正确显示

## 🆘 常见问题

### Q1: 照片不显示怎么办？
**A:** 检查以下几点：
1. 文件名是否完全正确（区分大小写）
2. 文件是否放在正确的文件夹中
3. 文件格式是否为 `.jpg` 或 `.png`
4. 按 `F12` 打开浏览器控制台，查看是否有错误提示
5. 确认照片文件不是 0KB（损坏的文件）

### Q2: 可以用其他文件名吗？
**A:** 可以，但需要修改 HTML 代码中的文件名。建议直接使用指定的文件名更简单。

### Q3: 照片显示变形怎么办？
**A:**
- **个人头像：** 使用正方形照片（1:1 比例）
- **奖项照片：** 使用 4:3 比例的照片
- 如果照片比例不对，建议用图片编辑软件裁剪

### Q4: 我没有某些奖项的照片怎么办？
**A:**
- 可以使用奖状扫描件
- 可以使用相关活动的现场照片
- 可以使用项目成果展示照片
- 暂时没有照片可以先用占位图，后续再替换

### Q5: 可以用 PNG 格式吗？
**A:** 可以！PNG 和 JPG 都支持。如果使用 PNG，确保文件名扩展名是 `.png`，并相应修改 HTML 中的引用。
